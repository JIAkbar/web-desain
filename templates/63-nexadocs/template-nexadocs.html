Instalasi NexaDocs

Pasang NexaDocs SDK ke proyek Anda dalam hitungan detik. Tersedia untuk Node.js, Python, dan Go. 

npm install nexadocs-sdk

import { NexaDocs } from 'nexadocs-sdk'

const docs = new NexaDocs({
  apiKey: 'nd_sk_xxxx'
})
