Instalasi DocsHub

Sebelum memulai, pastikan Anda sudah menginstal Node.js versi 18 ke atas dan npm atau yarn sebagai package manager.

Instal DocsHub secara global menggunakan npm, kemudian inisialisasi proyek baru Anda dengan satu perintah:

# Install DocsHub CLI secara global
npm install -g @docshub/cli

# Inisialisasi proyek baru
docshub init my-docs
cd my-docs
docshub dev

// Inisialisasi DocsHub Client import { DocsHub } from '@docshub/sdk'; const client = new DocsHub({ apiKey: 'dh_live_your_api_key_here', version: 'v2', timeout: 5000, }); // Ambil dokumen berdasarkan slug async function getDocument(slug) { try { const doc = await client.docs.get(slug); console.log(`Judul: ${doc.title}`); return doc; } catch (err) { console.error('Error:', err.message); throw err; } } // Cari dokumen dengan full-text search const results = await client.search.query({ q: 'instalasi konfigurasi', limit: 10, lang: 'id', });
# Instalasi: pip install docshub-sdk from docshub import DocsHubClient from docshub.exceptions import DocsHubError # Inisialisasi client client = DocsHubClient( api_key="dh_live_your_api_key_here", version="v2", ) # Ambil dokumen berdasarkan slug def get_document(slug: str) -> dict: try: doc = client.docs.get(slug=slug) print(f"Judul: {doc['title']}") return doc except DocsHubError as e: print(f"Error: {e.message}") raise # Cari dokumen dengan filter bahasa results = client.search.query( q="instalasi konfigurasi", limit=10, lang="id", ) print(f"Ditemukan: {results['total']} dokumen")

Fitur DocsHub

Dibangun untuk developer yang menghargai kecepatan, keterbacaan, dan kemudahan navigasi.

🔍

Full-text Search

Cari konten apa pun di seluruh dokumentasi dalam milidetik. Didukung algoritma pencarian canggih dengan dukungan bahasa alami dan highlighting hasil pencarian.

🌐

Multi-bahasa

Dokumentasi tersedia dalam 15 bahasa termasuk Bahasa Indonesia, Inggris, Mandarin, Jepang, dan lainnya. Kontribusi terjemahan terbuka untuk komunitas.

🌙

Dark Mode

Tampilan gelap yang nyaman untuk mata, dengan kontras yang dioptimalkan dan transisi warna yang mulus. Preferensi disimpan otomatis di browser Anda.

📋

Versi Kontrol

Akses dokumentasi untuk setiap versi produk Anda. Navigasi antar versi dengan mudah, dan lihat perubahan yang terjadi di setiap rilis secara jelas.
