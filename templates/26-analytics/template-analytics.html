Pertanyaan Teknis
yang Sering Ditanyakan

Berapa lama waktu yang dibutuhkan untuk setup dan onboarding? +
Setup rata-rata membutuhkan waktu 15–30 menit untuk koneksi sumber data pertama. Kami menyediakan guided wizard step-by-step, dokumentasi lengkap, dan video tutorial. Tim onboarding kami juga siap membantu via live chat selama jam kerja.
Bagaimana QuantumGrid menangani volume data yang sangat besar? +
QuantumGrid menggunakan arsitektur columnar storage berbasis Apache Parquet dan engine query terdistribusi. Kami mampu memproses puluhan miliar baris per query. Plan Enterprise mendukung partitioned tables dan clustering untuk performa optimal.
Apakah data saya aman? Bagaimana enkripsi dan compliance-nya? +
Semua data dienkripsi at-rest (AES-256) dan in-transit (TLS 1.3). Kami beroperasi di data center tier-3 di Indonesia dan mematuhi regulasi PDPA. Audit log lengkap tersedia. Plan Enterprise mendukung on-premise deployment untuk data yang sangat sensitif.
Bisakah saya membuat custom metric dan formula spesifik untuk bisnis saya? +
Ya, QuantumGrid mendukung Custom Calculated Fields menggunakan sintaks SQL standar atau formula builder visual. Anda bisa mendefinisikan KPI khusus, menggabungkan metrik lintas sumber data, dan menyimpannya sebagai metric library yang reusable di semua dashboard.
Apakah ada limit API? Bagaimana rate limiting-nya bekerja? +
Plan Growth mendapat 50.000 API request/bulan dengan rate limit 100 req/menit. Plan Enterprise mendapatkan unlimited request dengan rate limit yang dapat dikonfigurasi sesuai kebutuhan. Semua endpoint menggunakan OAuth2 bearer token. Dokumentasi lengkap tersedia di docs.quantumgrid.id.
