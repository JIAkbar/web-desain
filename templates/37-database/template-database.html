Pertanyaan yang Sering Ditanyakan

Tidak menemukan jawaban yang Anda cari? Tim developer kami siap membantu via Discord atau email.

Apakah bisa menulis query SQL custom di DataSphere? +
Ya, paket Pro ke atas memiliki SQL Mode penuh dengan editor yang dilengkapi syntax highlighting, autocomplete, dan explain plan. Anda juga bisa membuat view, function, dan trigger langsung dari antarmuka.
Bagaimana format endpoint REST API yang dihasilkan? +
Setiap tabel mendapat endpoint RESTful standar: GET /api/[tabel], POST /api/[tabel], PATCH /api/[tabel]/{id}, DELETE /api/[tabel]/{id}. Mendukung filter (?kolom=nilai), sort (?sort=kolom&order=asc), dan pagination (?page=1&limit=50). Dokumentasi Swagger tersedia otomatis.
Seberapa sering backup dilakukan dan berapa lama tersimpan? +
Paket Gratis: backup harian, tersimpan 7 hari. Paket Pro: backup setiap 6 jam, tersimpan 30 hari dengan point-in-time recovery. Paket Team ke atas: backup setiap jam, tersimpan 365 hari. Enterprise dapat dikonfigurasi sesuai kebutuhan.
Berapa ukuran database maksimum yang bisa ditampung? +
Paket Free: maksimum 500 MB. Paket Pro: 50 GB. Paket Team: 500 GB. Enterprise tidak terbatas (tergantung kontrak). Untuk dataset sangat besar (Terabyte), kami menyediakan solusi DataSphere Warehouse yang dioptimalkan untuk analitik.
Apakah data saya bisa diekspor jika ingin pindah platform? +
Tentu. Anda bisa ekspor data kapan saja dalam format CSV, Excel, JSON, atau SQL dump (PostgreSQL-compatible). Tidak ada vendor lock-in. Ekspor lengkap termasuk schema tersedia di semua paket termasuk Gratis.
