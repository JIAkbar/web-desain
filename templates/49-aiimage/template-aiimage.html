Pertanyaan Umum

Ada yang Ingin Ditanyakan?

Jawaban untuk pertanyaan yang paling sering muncul.

Siapa yang memiliki hak cipta gambar yang saya buat? +
Kamu memiliki hak penuh atas semua gambar yang dibuat menggunakan akun PixelMind-mu. Untuk paket Pro dan Creator, gambar dapat digunakan untuk keperluan komersial termasuk dijual, diprint, atau dipublikasikan tanpa royalti tambahan. Paket Free termasuk watermark PixelMind.
Apakah ada batasan konten yang tidak boleh dibuat? +
Ya. PixelMind memiliki sistem filter otomatis yang mencegah pembuatan konten kekerasan, konten seksual eksplisit, pelecehan, atau konten yang melanggar hak cipta pihak lain. Setiap prompt diproses melalui safety layer sebelum generasi dimulai. Pelanggaran berulang akan mengakibatkan suspensi akun.
Bagaimana kualitas gambar dibandingkan Midjourney atau DALL-E? +
PixelMind v3.0 menggunakan arsitektur yang setara dengan model terkini di industri, dengan keunggulan spesifik di pemahaman konteks bahasa Indonesia dan referensi budaya lokal. Untuk subjek seperti batik, wayang, atau lanskap Nusantara, PixelMind secara konsisten menghasilkan kualitas yang lebih relevan secara kultural.
Format apa saja yang tersedia untuk download? +
Semua paket mendukung download PNG dan JPG. Paket Pro dan Creator juga mendukung WebP (lebih kecil untuk web), serta PNG transparan (background transparan setelah background remover). Format RAW tersedia sebagai beta feature untuk paket Creator.
Apakah API tersedia untuk integrasi ke aplikasi saya? +
Ya! REST API tersedia untuk semua pengguna terdaftar. Paket Free mendapat 100 API request/bulan untuk testing. Paket Pro mendapat 10.000 request/bulan, dan Creator mendapat 50.000 request/bulan. Dokumentasi lengkap, SDK untuk Node.js, Python, dan PHP, serta sandbox environment gratis tersedia di developer.pixelmind.id.
