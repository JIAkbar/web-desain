Pertanyaan yang Sering
Ditanyakan Tim IT

Tidak menemukan jawaban yang Anda cari? Hubungi tim sales engineering kami.

Bagaimana proses migrasi data dari sistem lama (SAP, Oracle, atau Excel)? +
Tim data engineer kami menyediakan script migrasi untuk 30+ format sistem ERP populer. Untuk Excel, kami memiliki template import standar. Data divalidasi dalam tiga tahap: struktur, duplikasi, dan integritas relasional. Seluruh proses migrasi rata-rata memakan waktu 5–10 hari kerja tergantung volume data dan kompleksitas chart of accounts Anda.
Apakah NexaERP bisa di-host di server internal perusahaan (on-premise)? +
Ya, paket Enterprise mendukung deployment on-premise, hybrid cloud, maupun private cloud di AWS Jakarta, GCP Jakarta, atau Azure Southeast Asia. Untuk paket Starter dan Business, kami menggunakan infrastruktur cloud kami sendiri dengan data center di Jakarta (Tier-3 certified) dengan uptime SLA 99.9%. Backup otomatis harian dengan retensi 90 hari.
Apa saja sistem eksternal yang bisa diintegrasikan dengan NexaERP? +
NexaERP menyediakan REST API lengkap dan webhook untuk integrasi dua arah. Koneksi siap pakai (plug-and-play) tersedia untuk: Tokopedia, Shopee, Lazada, TikTok Shop, Bukalapak (marketplace); Midtrans, Xendit, DOKU (payment gateway); Anteraja, JNE, JET, SiCepat (logistik); serta sistem absensi Fingerspot, Hikvision, dan ZKTeco. Untuk kebutuhan kustom, tim API kami siap membantu.
Bagaimana skema dukungan teknis dan SLA support NexaERP? +
Starter: support via email dan tiket, respon maks 24 jam (hari kerja). Business: WhatsApp, email, dan telpon, respon maks 2 jam 24/7/365, dengan dedicated account manager. Enterprise: engineer on-call 24/7 dengan respon 30 menit untuk critical issues, serta dedicated customer success manager yang melakukan review bulanan performa sistem Anda.
Bagaimana NexaERP memastikan keamanan data sensitif perusahaan kami? +
NexaERP bersertifikat ISO 27001 dan SOC 2 Type II. Semua data dienkripsi AES-256 saat penyimpanan dan TLS 1.3 saat transmisi. Audit trail lengkap untuk setiap aksi pengguna. Fitur keamanan termasuk: 2FA wajib, IP whitelist, session management, RBAC granular, dan penetration testing rutin oleh pihak ketiga independen setiap 6 bulan. Data Anda tidak pernah digunakan untuk keperluan lain.
